Key Design Decisions

Domain Layer Isolation

Pure TypeScript with no React/storage dependencies. Enables future native apps and backend migration.

Timer Engine Separation

Core timer logic independent of UI. Handles phase transitions, pause/resume, and drift correction.

Store-per-Concern

Separate Zustand stores for workouts, timer state, and sessions. Clean boundaries, easy testing.

Persistence Abstraction

Dexie wrapper allows swapping to API calls later without changing store logic.
